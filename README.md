# Phosphor

This is a bit of a fork of [crush](https://github.com/charmbracelet/crush/). The changes made here are very much likely not compatible with that project. The fork started from wanting to use crush with some local models. While crush supports locally hosted models, it wasn't working well with certain setups and would end up in a lot of tool call failures and loops. So some changes were made to work better with those models and inference engines.

Then there were some changes to add some additional functionality, such as tree-sitter, that went well beyond the scope of the original project. There are some additional planned features that are going to be even farther off course. So the responsible thing to do was a proper fork, and here we are.
